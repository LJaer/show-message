##3、配置
###a、下载源码后，导入myeclipse2014，将sql文件夹下的sql文件导入mysql数据库
###b、修改config文件夹下db.properties文件，根据实际情况修改
###c、在浏览器输入http://localhost:8080/show-message/login进入后台登录界面，账户名admin，密码admin
###d、在浏览器输入http://localhost:8080/show-message/index.jsp进入网站首页
